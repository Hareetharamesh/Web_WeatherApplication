# Weather Application (HTML / CSS / JavaScript)

A responsive weather dashboard built using the OpenWeather REST API.  
Displays real-time temperature, humidity, wind speed, and weather conditions.

## Features
- Real-time weather data fetching using REST API
- JSON parsing & error handling
- Responsive UI with dynamic icons
- API integration using Fetch()
- Clean and minimal UI design

## Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeather API

## How to Run
1. Download or clone the repo
2. Open index.html in any browser
